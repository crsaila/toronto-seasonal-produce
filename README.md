# Toronto seasonal produce calendar

An interactive Gantt-style chart showing when ~70 fruits and vegetables are in season in the Toronto / southern Ontario area, week by week across the full year.

## Features

- 52-week grid per produce item, colour-coded by season type (peak, shoulder, greenhouse, storage)
- "Now" marker showing the current week
- Hover to highlight a week column with a tooltip showing what's in season
- Row hover highlighting
- Filter by type (fruit/vegetable), source (field, greenhouse, foraged, storage), search by name, sort
- Regional highlights — 13 items uniquely tied to the Toronto/southern Ontario foodshed
- Works in Safari and all modern browsers

## Usage

Open `index.html` directly in a browser — no build step, no dependencies.

## Files

- `index.html` — the application (self-contained HTML/CSS/JS)
- `BRIEF.md` — full project brief including data sources, colour scheme, data structure, and Safari compatibility notes

## Data sources

Synthesised from Foodland Ontario (Government of Ontario), Ontario Greenhouse Vegetable Growers, and regional farmers' market reports. See `BRIEF.md` for full sourcing details.
